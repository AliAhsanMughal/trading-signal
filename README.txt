# NZD/CAD Trading Signal App for Windows

## ✅ Features
- RSI + MACD-based signals
- 1-minute updates
- Windows desktop GUI (Tkinter)

## ▶️ How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the app:
   ```
   python app.py
   ```

3. (Optional) Convert to .exe using PyInstaller:
   ```
   pip install pyinstaller
   pyinstaller --onefile --windowed app.py
   ```

This will create a standalone `.exe` in the `/dist` folder.
